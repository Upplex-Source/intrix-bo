Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://localhost

Authenticating requests

This API is not authenticated.

Cart API

1. Add to cart

Example request:
curl --request POST \
    "http://localhost/api/v1/carts/add" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"product_code\": \"5-IN-1\",
    \"color\": \"CHROME\",
    \"quantity\": \"1\",
    \"session_key\": \"kn1i23onlas1\",
    \"promo_code\": 0
}"

const url = new URL(
    "http://localhost/api/v1/carts/add"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "product_code": "5-IN-1",
    "color": "CHROME",
    "quantity": "1",
    "session_key": "kn1i23onlas1",
    "promo_code": 0
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

2. Retrieve cart

id and session_key can be used to filter out the cart
Example request:
curl --request GET \
    --get "http://localhost/api/v1/carts?session_key=abcd-1234&id=abcd-1234&per_page=10" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/v1/carts"
);

const params = {
    "session_key": "abcd-1234",
    "id": "abcd-1234",
    "per_page": "10",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Promo Code API

1. Get promo code

requires authentication

Get all voucher that is claimable

voucher discount type
1: percentage
2: fixed amount
3: buy x free y

Example request:
curl --request GET \
    --get "http://localhost/api/v1/promo-codes?per_page=10&promo_code=XBMSD22&discount_type=2" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/v1/promo-codes"
);

const params = {

    "promo_code": "XBMSD22",

};
Object.keys(params)

const headers = {
